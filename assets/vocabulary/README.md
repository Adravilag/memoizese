# English Vocabulary Tests - B1/B2 Level

Este directorio contiene **20 archivos de vocabulario** en inglés organizados por temas, diseñados para nivel B1-B2 (intermedio a intermedio-alto).

## Estructura

Cada archivo contiene **40 preguntas de opción múltiple** con el siguiente formato:
- Pregunta sobre el significado de una palabra o expresión
- 4 opciones (a, b, c, d)
- Respuestas correctas al final con `[BOLD_ANSWERS: ...]`

## Formato de las preguntas


## Nivel

- **B1** (Intermediate): Vocabulario frecuente y expresiones comunes
- **B2** (Upper-Intermediate): Vocabulario más avanzado, phrasal verbs, collocations

## Tipos de palabras incluidas

- ✅ Sustantivos (nouns)
- ✅ Verbos (verbs)
- ✅ Adjetivos (adjectives)
- ✅ Expresiones idiomáticas (idioms)
- ✅ Phrasal verbs
- ✅ Collocations
